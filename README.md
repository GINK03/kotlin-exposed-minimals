# Kotlin Exposed

KotlinのSQL言語をDSLとして利用できるライブラリ。  

便利で、DBと接続して何かやるときには、必須アイテムなのではいかと思う  


## gradle options 

**build**  
```console
$ ./gradlew build
```

**run** 
```console
$ ./gradlew run
```

**stand-alone jar**  
```console
$ ./gradlew jar
```

**stand-alone jarの実行**  
```console
$ kotlin -cp ./build/libs/kotlin-exposed-minimals.jar HelloWorldKt
```
